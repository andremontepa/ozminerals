Descrição dos fontes

A103CND2.prw - Ponto de Entrada para gravar no Protheus o personalizado 2 (Data e valores digitados pelos usuários).
MT116AGR.prw - Ponto de Entrada para gravar no Protheus os dados complementares do CTe gerados pela rotina Mata116.
MT116GRV.prw - Ponto de Entrada para gravar chave e tipo do Cte nas tabelas SF3 e SFT do Protheus.
MTUFOPRO_2021.prw - Programa que gera os documentos fiscais no Protheus, seja Notas Fiscais de Produto, Serviço e Cte pela 
rotina Mata103 ou Cte pela rotina Mata116.
upd_znf.prw - Programa que cria a tabela ZNF na base do cliente, nome do update a ser executado U_UPDEXPZNF. Antes de enviar 
para o cliente deverá ser confirmado o tamanho do campo filial das tabelas do cliente, após confirmar, alterar o tamanho 
do campo na linha 525 deste fonte. Existe um fonte especifico para quando o dicionário do Protheus estiver no banco de dados e quando for SX, se atentar a isso.
Exemplo: 2																, ; //X3_TAMANHO tamanho 2
Updexp_F1_USUSMAR - Programa que cria o campo F1_USUSMAR na tabela SF1. Nome do update a ser executado U_UPD_ABAXF1. Existe um fonte especifico para quando o 
dicionário do Protheus estiver no banco de dados e quando for SX, se atentar a isso.
WSCFGTable_3_NEW_NOVO - Programa para consultar o banco do cliente via serviço de Webservices do Protheus, só utilizar quando
o Ábax e o Protheus estiverem em redes diferentes sem possibilidade de conexão direta via banco. Dependendo da situação poderá
deixar o acesso lento.
